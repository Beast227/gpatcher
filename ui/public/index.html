Quick Reference Commands

Command Purpose
gpatcher create Produce a delta patch zip between two game installations.
gpatcher apply Upgrade an old game installation using a delta patch.
gpatcher restore Revert a patch using the auto-created backup files.
gpatcher search Search the Internet Archive database for available game patches.
gpatcher update Download and install the latest gpatcher tool release from GitHub.
